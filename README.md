# Journey to a performant website

Slides and demo for my "Journey to a performant website" talk. 

To view the slides and demo:

1. Install dynassets

```bash
$ npm i -g dynassets
```

2. start dynassets server

```bash
$ dynassets
```

3. Start serving root directory 

```bash
$ npm start
```

4. Open url in your browser

`http://localhost:9000`
